# Social Media Qt App

A desktop social media application built with **C++**, **Qt Widgets**, and **Visual Studio**.

## Features
- Login window
- Custom UI (Qt Designer)
- Resource system with `.qrc`
- Modular project structure

## Tech Stack
- C++
- Qt 6
- Visual Studio
- Qt VS Tools

## Project Structure
src/        → C++ source files  
include/    → header files  
forms/      → Qt Designer UI files  
resources/  → Qt resource file and assets  

## Build & Run
1. Open `Social Media project.sln` in Visual Studio  
2. Select **x64** and **Debug** or **Release**  
3. Build and run  

If you add new headers, make sure the `include/` directory is listed in:  
Project Properties → C/C++ → Additional Include Directories
